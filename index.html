Constantes, Matrices y Switch / Case.

Definición de constante.

Las constantes son identificadores (nombres) que representan valores que no pueden cambiar durante la ejecución del script. A diferencia de las variables, las constantes no pueden cambiar su valor una vez que se les ha asignado. Son útiles para almacenar valores que no deben modificarse a lo largo del programa y se utilizan para mejorar la legibilidad y mantenibilidad del código.

Sintaxis básica.

// Las constantes asi como en Javascript tienen su 
// propia palabra reservada. En este caso es la palabra
// "define". 

//Las constantes estan compuestas por 2 parametros
				// El nombre     // Valor
define("NOMBRE_CONSTANTE", "valor");

// El nombre puede ser cualquiera que nosotros queramos poner
// El valor ingresado puede contener ya sea tipo numerico, 
// flotante, entero o caracter.

Ejemplo de sintaxis:

// EJEMPLO PARA CONECTAR A UNA BASE DE DATOS
define("DB_HOST", "localhost");
define("DB_USUARIO", "usuario");
define("DB_CONTRASEÑA", "contraseña");
define("DB_NOMBRE", "basededatos");
// EJEMPLO PARA DEFINIR LA CONSTANTE "PI"
define("PI", 3.14159);
echo "El valor de PI es: " . PI;
// EJEMPLO DONDE DEFINIMOS LOS DIAS.
define("LUNES", "Lunes");
define("MARTES", "Martes");
define("MIERCOLES", "Miércoles");
define("JUEVES", "Jueves");
define("VIERNES", "Viernes");

Ejemplo practico

// EJEMPLO PRACTICO EN DONDE DETERMINAMOS LA CONVERSION DE TEMPERATURAS
<?php
// Definimos las constantes a utilizar
// factor de conversión representa el valor numérico
// o la proporción que se utiliza para relacionar 
// una unidad de medida con otra
define("FACTOR_CONVERSION", 9/5);
// Es la constante 32 en la fórmula. Representa la 
// diferencia de temperatura entre el punto de congelación
// del agua en grados Fahrenheit (32°F) y el punto de congelación
// del agua en grados Celsius (0°C).
define("OFFSET_CONVERSION", 32);

// Declaramos la variable que va a contener nuestro valor en
// en temperatura Celcius ingresada por el usuario.
$temperaturaCelsius = 25;

// Definimos la variable que va a contener la operacion y resultado 
// 25 x (9/5 + 32)
$temperaturaFahrenheit = $temperaturaCelsius * FACTOR_CONVERSION + OFFSET_CONVERSION;

// Se imprime el resultado en pantalla de la operacion pasada.
echo "$temperaturaCelsius °C es igual a $temperaturaFahrenheit °F";
?>

Definición de Matriz.

Una matriz es una estructura de datos que puede contener varios valores o elementos. Se puede ver como una colección de elementos dispuestos en filas y columnas. Cada elemento en una matriz tiene una posición única, especificada por un conjunto de índices.

Representación de Matrices en PHP.

Matrices Indexadas:

las matrices indexadas se utilizan cuando necesitas almacenar datos de manera estructurada y acceder a ellos mediante un índice numérico.

Utilidad:

Listas o colecciones ordenadas: Almacenar y acceder a elementos en una lista o colección ordenada, como una lista de compras o nombres de estudiantes.

$shoppingList = array("Apples", "Bananas", "Bread");
echo $shoppingList[1]; // Muestra "Bananas"

Iteración a través de elementos: Cuando necesitas recorrer y manipular cada elemento de una lista de manera secuencial.

$colors = array("Red", "Green", "Blue");
foreach ($colors as $color) {
    echo $color . " ";
}
// Muestra: Red Green Blue

Almacenamiento de datos tabulares simples: Para representar datos tabulares simples, como una matriz bidimensional para almacenar información de estudiantes (por ejemplo, nombres y edades).

$students = array(
    array("John", 20),
    array("Jane", 22),
    array("Doe", 21)
);

echo $students[0][0]; // Muestra "John"

Manejo de datos secuenciales: Cuando necesitas mantener un orden específico para los datos y acceder a ellos de acuerdo con su posición.

$months = array("January", "February", "March");
echo $months[2]; // Muestra "March"

Sintaxis básica.

  • Cada elemento de la matriz tiene un índice numérico.
  • El índice comienza desde 0 y se incrementa en 1 para cada elemento.
$matrizIndexada = array(valor1, valor2, valor3);
// Construccion de una matriz indexada
$matrizIndexada = array("Manzana", "Banana", "Cereza");

Ejemplo practico.

//Cálculo con Matrices Indexadas
// Aquí se define una matriz bidimensional
// llamada $matriz. Esta matriz tiene 3 filas y 3 columnas
// y cada elemento de la matriz es un número entero.

$matriz = array(
    array(1, 2, 3),
    array(4, 5, 6),
    array(7, 8, 9)
);

// 1.- Se inicializa una variable $sumaDiagonal en 0.
// 2.- Luego, hay un bucle for que itera sobre las filas de
// la matriz.
// 3.- En cada iteración, se agrega el valor de la
		// posición actual en la diagonal principal (la posición $i, $i)
		// a la variable $sumaDiagonal.
$sumaDiagonal = 0;
for ($i = 0; $i < count($matriz); $i++) {
    $sumaDiagonal += $matriz[$i][$i];
}
// Finalmente, se imprime en pantalla el resultado de
// la suma de la diagonal principal.
echo "La suma de la diagonal principal es: $sumaDiagonal";

Matrices Asociativas:

Una matriz asociativa es un tipo de matriz en la que cada elemento está asociado con un nombre o una clave en lugar de un índice numérico. Cada par clave-valor en la matriz está formado por dos partes: la clave que actúa como un identificador y el valor asociado a esa clave. Esto proporciona una forma más descriptiva de trabajar con datos en comparación con las matrices indexadas.

Sintaxis basica.

// Cada elemento tiene un par clave-valor,
// donde la clave es una cadena que actúa como un identificador único.

// Este tipo de matriz es similar a un
// diccionario en otros lenguajes de programación.
$matrizAsociativa = array("clave1" => valor1, "clave2" => valor2, "clave3" => valor3);
  • Cada elemento tiene un par clave-valor, donde la clave es una cadena que actúa como un identificador único.
  • Este tipo de matriz es similar a un diccionario en otros lenguajes de programación.
$persona = array(
    "nombre" => "Juan",
    "edad" => 25,
    "ciudad" => "Ejemplo City"
);

Cuándo usar Matrices Asociativas:

Datos estructurados: Utiliza matrices asociativas cuando necesites representar datos estructurados con etiquetas descriptivas. Por ejemplo, información de un usuario con claves como "nombre", "edad" y "correo electrónico".

$usuario = array(
    "nombre" => "Ana",
    "edad" => 30,
    "email" => "ana@example.com"
);

Configuraciones y opciones: Cuando estás trabajando con configuraciones o opciones que tienen nombres descriptivos. Esto facilita la comprensión del propósito de cada configuración.

$configuracion = array(
    "color_fondo" => "azul",
    "tamano_fuente" => 14,
    "tema" => "claro"
);

Respuestas de bases de datos: Cuando recuperas datos de una base de datos y cada campo tiene un nombre asociado. Las claves en la matriz asociativa pueden coincidir con los nombres de las columnas en la base de datos.

$fila = array(
    "id" => 1,
    "nombre" => "Carlos",
    "puntuacion" => 85
);

Iteración con claves descriptivas: Útil cuando necesitas iterar a través de la matriz y quieres acceder tanto a las claves como a los valores.

foreach ($usuario as $clave => $valor) {
    echo $clave . ": " . $valor . "<br>";
}
// Muestra:
// nombre: Ana
// edad: 30
// email: ana@example.com

Switch / Case.

Switch y Case son estructuras de control en PHP que permiten realizar acciones diferentes según el valor de una expresión. Es una alternativa más limpia y eficiente a una serie de declaraciones if-else anidadas cuando se desea comparar una variable con varios valores posibles.

Sintaxis basica.

$variable = valor_a_evaluar;

switch ($variable) {
    case valor1:
        // Acciones a realizar si $variable es igual a valor1
        break;
    case valor2:
        // Acciones a realizar si $variable es igual a valor2
        break;
    // Puedes tener tantos casos como desees
    default:
        // Acciones a realizar si ninguno de los casos anteriores coincide
}

Funcionamiento:

  1. La expresión entre paréntesis después de switch se evalúa una vez.
  1. Cada case compara el valor de la expresión con el valor del case.
  1. Si hay una coincidencia, se ejecutan las acciones dentro de ese case.
  1. La palabra clave break se usa para salir del switch después de que se ejecuta un case.
  1. Si no hay coincidencias, se ejecuta el default (si está presente).

Cuándo Usar:

  • Múltiples condiciones para una variable: Cuando necesitas comparar una variable con varios valores y realizar acciones diferentes según el valor.
  • Código más legible: Puede hacer que tu código sea más fácil de entender en comparación con múltiples declaraciones if-else.

Ejemplo practico.

$opcion = 2;

switch ($opcion) {
    case 1:
        echo "Seleccionaste la opción 1.";
        break;
    case 2:
        echo "Seleccionaste la opción 2.";
        break;
    case 3:
        echo "Seleccionaste la opción 3.";
        break;
    default:
        echo "Opción no válida.";
}

Dependiendo del valor de $opcion, se ejecutará el bloque de código correspondiente. Es especialmente útil cuando tienes varias opciones y no deseas escribir una serie de if-else consecutivos.
